# iDragon 🐉

A simple JavaScript game where a dino jumps to avoid dragons!  
Built using HTML, CSS, and JavaScript.

### 🔊 Features:
- Jump mechanic with arrow keys
- Background music & game over sound
- Collision detection
- Score tracking
- Responsive to key events
